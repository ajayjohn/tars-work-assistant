# Discovery Reference

Use before generating ideas when the topic is ambiguous or context-dependent.

1. Resolve the subject, audience, constraint, and desired output.
2. Pull a bounded context bundle for relevant people, initiatives, decisions,
   products, and recent journal entries.
3. Identify what is fixed, what can vary, and what would make an idea useful.
4. Ask at most one clarifying question only if the missing answer materially
   changes the generation space.

Discovery output should be compact enough to hand to generation subagents.

# Analyze Reference

Use for strategy, tradeoffs, risk reviews, and executive decision support.

1. Build a bounded context bundle for the topic.
2. State the decision, constraint, or tension in one sentence.
3. Separate known facts, assumptions, unknowns, and judgment calls.
4. Compare options against the user's actual operating context.
5. End with a recommendation, key risks, and the smallest useful next step.

Do not persist decisions or tasks without review.

# Daily Briefing Reference

## Gather

- Calendar events for target date
- Tasks due or overdue
- Workspace map: active initiatives, inbox count, recent journal
- Context gaps
- People profiles for high-priority attendees

## Shape

Start with the single most important thing. Then include schedule, priority
tasks, meeting prep, initiative pulse, gaps, and one next ask if needed.

Save as `journal/YYYY-MM/YYYY-MM-DD-daily-briefing.md` unless this is purely a
re-entry orientation with no new durable content.

# Adaptive Briefing Reference

Use this before selecting daily or weekly detail.

## Inputs

- `mcp__tars_vault__context_gaps`
- `mcp__tars_vault__workspace_map`
- Calendar and task capabilities when connected

## Postures

| Posture | Signal | Output adjustment |
|---|---|---|
| Current | recent TARS use and recent intake | normal focus and prep |
| Re-entry | long gap since last session or "catch me up" | what changed, stale, unknown |
| Sparse context | calendar/tasks but little recent journal/transcript | show uncertainty and ask for one missing input |
| Drift-aware | stale active initiatives, overdue tasks, inbox backlog | prioritize drift repair |

End with one concrete ask when TARS lacks enough context.

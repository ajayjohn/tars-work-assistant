# Weekly Briefing Reference

## Gather

- Calendar range for the week
- Tasks due, overdue, delegated, and stale backlog
- Active initiatives and milestones
- Recent journal and decisions
- Context gaps and system state

## Shape

Summarize last week, this week, open tasks, initiative health, people context,
focus time, and stale context. Save as
`journal/YYYY-MM/YYYY-MM-DD-weekly-briefing.md`.

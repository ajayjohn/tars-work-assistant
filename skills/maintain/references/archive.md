# Archive Maintenance Reference

Archive is active-set management. It keeps the working set around 2,000 active
files while preserving history.

Use `mcp__tars_vault__archive_candidates` or `scripts/archive.py --json` for
dry-run candidates. Protect pinned notes, durable decisions, org context, active
initiatives, active-task references, and recent backlinks. Archive never
deletes and never runs without review.
